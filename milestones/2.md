## Milestone: Minimal Graph Runtime

- Graph runtime:
  - Node representation with op type, named inputs, and named output
  - Tensor table based on named tensors
  - Topological execution from unordered node insertion
  - Graph dump, execution order dump, and tensor table dump
  - Add operator and residual / skip connection graph
  - Graph validation for duplicate node names, duplicate outputs, wrong input count, missing dependencies, cycles, and missing requested outputs
